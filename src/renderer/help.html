Setup Required: Enable Audit Events

Screen time tracking requires Windows Audit Logon Events to be enabled. If you're seeing "No activity tracked yet", follow these steps:

Step 1: Enable "Audit Logon Events"

  1. Press Win + R, type secpol.msc, press Enter
  2. Navigate to: Local Policies → Audit Policy
  3. Double-click Audit Logon Events
  4. Check both: Success and Failure
  5. Click OK

Step 2: Enable "Audit Other Logon/Logoff Events"

  1. Press Win + R, type gpedit.msc, press Enter
  2. Navigate to:
    Computer Configuration → Windows Settings → Security Settings → Advanced Audit Policy Configuration
  3. Find and double-click Audit Other Logon/Logoff Events
  4. Enable both Success and Failure

Step 3: Apply Changes

Run Command Prompt as Admin and execute:

gpupdate /force

Step 4: Run as Administrator

Make sure to run this app as Administrator to read audit logs.

Note: Lock/unlock your computer after enabling these settings to start tracking.
